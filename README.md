# 🌯 Next Snack

**Next Snack** is a simple food ordering web app inspired by **Dodo Pizza**, but with snacks instead of pizza 😄.

The project demonstrates a modern food catalog interface with category navigation, smooth scrolling, and a clean UI built with the Next.js ecosystem.

---

## 🚀 Description

Next Snack allows users to browse snacks by categories and view products in a fast and responsive interface.  
The UI highlights the active category while scrolling, similar to popular food delivery apps.

This project was built mainly to practice modern **Next.js App Router architecture** and frontend patterns.

---

## 🛠 Tech Stack

1. **NextJS** (Parallel Routes, Group Routes, Server Actions, API)
2. **TypeScript**
3. **TailwindCSS + ShadCN**
4. **Prisma + PostgreSQL**
5. **React Hook Form + Zod**
6. **Zustand**
7. **react-use**
8. **nextjs-toploader**
9. **react-hot-toast**
10. **lucide-react**
